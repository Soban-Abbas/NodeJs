# E-Commerce Web Application

A full-stack server-rendered e-commerce web application built with Node.js, Express.js, MongoDB, EJS, Authentication, Authorization, Sessions, Validation, and MVC Architecture.

This project is focused on learning backend engineering concepts step-by-step, including:

* RESTful API architecture
* Authentication & Authorization session based
* Error handling
* Validation
* MVC architecture
* MongoDB & Mongoose
* Protected routes
* Async programming
* Clean scalable backend structure

---
